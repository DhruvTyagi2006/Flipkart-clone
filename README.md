# Flipkart Clone (HTML & CSS)

A simple **Flipkart homepage clone** built using only **HTML and CSS**.  
This project recreates the look and feel of Flipkart’s landing page while focusing on layout design, responsiveness basics, and UI structuring using pure frontend technologies.

---

## Features

- Flipkart-style **navigation bar**
- **Search bar UI** similar to Flipkart
- **Product category section**
- **Product cards layout**
- **Banner / hero section**
- **Footer section**
- Clean and organized layout using **CSS styling**
- Fully built using **only HTML and CSS**

---

## Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout design  

---

## Project Structure

```
flipkart-clone/

├── index.html        # Main HTML file
├── style.css         # CSS styling file

└── README.md         # Project documentation
```

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/flipkart-clone.git
```

2. Open the project folder

3. Run the project by opening **index.html** in your browser.
